ETIQUETA CHARSET

Serveix per una modificació de la pàgina web.

ATRIBUT META NAME VIEWPORT

Serveix per treballar amb dissenys que s'han d'adaptar als mòbils. La majoria d'etiquetes meta tenen un nami i un content. El name fa referencia a quin tipus de metadades que s'han donat al navegador i quen és el cntingut d'quella metadada.

ETIQUETA META NAME DESCRIPTION

Porta un text com a contingut. Aquest text és el que apareix en aquest enllaç.

ETIQUETA META NAME AUTHOR

Porta un text com a contingut que explica qui és el propietari de la web o qui l'ha construit.
